JUnit-Conditional
===========
Adds conditional-run (or conditional-skip) to JUnit.
Inspired and many codes borrowed from [junit-ext](https://code.google.com/p/junit-ext/).

Available

- @RunIf
- @RunIfOs
- @RunIfSocketOpened
- @RunIfURLReachable

Future plan

- @RunIfFileExist
- @IgnoreIf~ (reverse of @RunIf~s)
- @RunAfter(date)/@IgnoreUntil(date)
